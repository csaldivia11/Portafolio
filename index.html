Proyectos Destacados

📚 Sistema de Gestión de Biblioteca Digital

Módulo 2 - Programación Orientada a Objetos

Descripción

Aplicación en Python para gestionar libros, usuarios y préstamos de la "Biblioteca Digital para Ingenieros de Datos" de DataSolutions Academy. El sistema permite registrar y consultar libros por autor, género y disponibilidad, administrar usuarios con diferentes roles (Estudiante, Instructor) y gestionar préstamos con validación completa.

Objetivos

Desarrollar una aplicación robusta aplicando funciones, estructuras de datos, POO y manejo de excepciones para automatizar la gestión de una biblioteca digital, eliminando procesos manuales propensos a errores.

Tecnologías Utilizadas

Python POO Herencia Polimorfismo Manejo de Excepciones

Características Clave

• Estructura modular organizada en paquetes
• Clases para Libros, Usuarios (Estudiante/Instructor) y Préstamos
• Sistema de excepciones personalizadas
• Validación de disponibilidad y control de errores
• Documentación completa con docstrings PEP8

Reflexión

Este proyecto me permitió consolidar los fundamentos de POO en Python. Aprendí la importancia de escribir código modular, documentado y robusto mediante el manejo adecuado de excepciones. La implementación de herencia y polimorfismo me ayudó a crear soluciones escalables y mantenibles.

🔄 Pipeline de Datos Analítico

Módulo 3 - Del Archivo Crudo al Reporte Analítico

Descripción

Pipeline completo de datos que simula un escenario empresarial real: procesamiento automatizado de archivos con errores, datos faltantes y formatos inconsistentes. El proyecto incluye generación de datos sintéticos (clientes.csv y ventas.xlsx), limpieza exhaustiva, detección y tratamiento de outliers, integración de fuentes, y generación de reportes analíticos.

Objetivos

Construir un pipeline profesional de data wrangling que automatice la transformación de datos crudos en información confiable para la toma de decisiones empresariales, aplicando técnicas avanzadas de limpieza y análisis.

Tecnologías Utilizadas

Python Pandas NumPy Matplotlib Data Wrangling

Características Clave

• Generación de datos sintéticos con errores intencionales
• Limpieza y normalización de columnas
• Tratamiento de valores nulos mediante imputación justificada
• Detección de outliers con IQR y z-score
• Integración de múltiples fuentes de datos
• Agrupamiento y resúmenes estadísticos con pivot tables
• Visualizaciones analíticas y exportación de reportes

Reflexión

Este proyecto me enseñó que los datos del mundo real son desordenados y requieren un proceso riguroso de limpieza. Aprendí a tomar decisiones fundamentadas sobre el tratamiento de nulos y outliers, y a documentar cada paso del proceso. La experiencia me preparó para enfrentar desafíos reales en proyectos de ingeniería de datos.

⚡ Flujos de Datos en Tiempo Real

Módulo 8 - Apache NiFi y Apache Kafka

Descripción

Implementación de arquitecturas de datos modernas para procesamiento batch y en tiempo real. El proyecto consta de dos tareas: (1) Flujo ETL con Apache NiFi que lee archivos CSV, transforma datos de CSV a JSON y carga en PostgreSQL, y (2) Sistema de streaming con Apache Kafka que incluye productor en Python simulando eventos de ventas y consumidor que procesa mensajes en tiempo real y almacena en SQLite.

Objetivos

Demostrar competencia en tecnologías de ingeniería de datos empresariales, implementando pipelines de ingesta batch con NiFi y streaming de eventos con Kafka, aplicando buenas prácticas de monitoreo y logging.

Tecnologías Utilizadas

Apache NiFi Apache Kafka Python PostgreSQL SQLite Zookeeper

Características Clave

Apache NiFi:
• Procesadores GetFile, UpdateAttribute, ConvertRecord, PutDatabaseRecord
• Transformación CSV a JSON con validación de esquemas
• Integración con PostgreSQL usando HikariCP Connection Pool
• Logging y monitoreo del procesamiento

Apache Kafka:
• Productor que simula eventos de sistema de ventas
• Consumidor con lógica de procesamiento en tiempo real
• Detección de clientes VIP con alertas automáticas
• Estadísticas en tiempo real y persistencia en SQLite

Reflexión

Este proyecto me introdujo a las tecnologías fundamentales de la ingeniería de datos moderna. Aprendí la diferencia entre procesamiento batch y streaming, y cómo cada uno tiene sus casos de uso específicos. La configuración de NiFi y Kafka me enseñó sobre arquitecturas distribuidas y la importancia del monitoreo en sistemas de datos en producción.
